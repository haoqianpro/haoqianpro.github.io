python内置的函数

一月 24, 2019
  • python

内置的类

  • slice，它会创建一个切片对象（是slice类的构造函数），可以用在任何可以进行切片操作的地方。

    • 对于一个切片对象实例s，可以通过s.start，s.stop，s.step来获得开始，结束，步长。
    • 可以使用indices(size)方法将切片映射到特定大小的序列上，防止越界。

  • complex，创建一个复数对象。

    • c.real提取实部，c.imag提取虚部，c.conjugate()获得共轭复数

  • range返回一个range对象，它是可迭代的

    当起始值未指定，默认是0，步长默认是1

  • reversed返回一个反向迭代器，只可作用于对象有确定的大小或实现了__reversed__()。

  • enumerate返回一个enumerate object

    返回(序号，元素)组成的元组，可指定序号起始值

  • zip同时对多个可迭代对象进行迭代。每次返回一个元组，当有一个对象迭代完成时结束。

内置函数

  • min，返回最小值，可以传入可调用对象指定比较关键字

  • max与min类似

  • sum返回从起始值累加的结果

  • any如果有可迭代对象中元素x使得bool(x)为真，则返回真。若对象为空，返回假

  • format格式化输出，指定宽度和精度的一般格式为[<>^]?width[,]?(.digits)?，同样适用于字符串的.format方法

  • round取整函数，用于将数字取整到固定小数位。当某个值恰好等于两个整数间的一半时，取整会取到离该值最接近的偶数上。当传递的ndigits为负数时，会相应的取整到十位，百位

  • bin，oct，hex会把整数转换为二进制，八进制，十六进制文本字符串形式。当不想出现相应的进制前缀时，可以使用format(x,'o')函数。将其他进制数转换为十进制数可使用int(x,base=10)并指明数值的进制。

  • len返回一个可迭代对象中元素的数量

  • next返回迭代器的下一个元素

    如果指定default则遇到迭代结束返回给定值代替StopIteration

  • iter

  • isinstance检查一个对象是否是一个类的实例或它的子类

    当给定一个元组时，会依次对元组中的元素进行检查。

  • open函数用于打开文件，返回一个文件对象。

  • print输出数据

    数据默认输出到标准输出也就是屏幕，当需要将输出重定向到文件中，只需要指定file关键字参数，文件要以文本模式打开
